# Racing Times Tracker

A web application framework for logging and tracking racing times with secure login functionality.

## Features

- **User Authentication**: Secure login system with session management
- **Racing Times Logging**: Add, edit, and delete racing lap times
- **Dashboard**: View all recorded racing times in an organized table
- **Responsive Design**: Works on desktop and mobile devices
- **Data Persistence**: Uses localStorage to store racing times

## Usage

1. Open `index.html` in a web browser
2. Log in with one of the demo credentials
3. Add racing times using the form on the dashboard
4. View, edit, or delete existing racing times

## Files Structure

- `index.html` - Login page
- `dashboard.html` - Main dashboard for racing times
- `styles.css` - Application styling
- `login.js` - Login functionality and authentication
- `dashboard.js` - Dashboard functionality for managing racing times

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- localStorage for data persistence
- sessionStorage for session management
